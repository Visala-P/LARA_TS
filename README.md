This repository contains the front-end for LARA, a low-cost adaptive robotic-AI therapy system for children.

Getting started (local)

1. Install dependencies:

```powershell
npm install
```

2. Start the dev server:

```powershell
npm run dev
```

This project is built with Vite, TypeScript, React, Tailwind CSS and shadcn-ui.

If you previously saw references to external tooling or services in this README (e.g. project generators), they have been removed. Work locally or in your preferred cloud IDE and push changes to your Git remote as usual.
